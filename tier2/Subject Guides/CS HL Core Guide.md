# CS HL – Core Guide

Deep dive into Computer Science Higher Level.

## Topic Overview

### Paper 1 & 2 Topics

1. **System fundamentals** – Hardware, software, database concepts
2. **Computer organization** – Architecture, memory, processing
3. **Networks** – Protocols, security, client-server
4. **Computational thinking** – Algorithm design, decomposition
5. **Programming** – Java, OOP, data structures

### Paper 3 Topics (HL only)

- Advanced topics: Resource optimization, advanced databases
- Code in context questions

## Key Concepts to Master

- [[Big O notation]] – Complexity analysis
- [[Recursion]]
- [[Linked lists]], [[Binary trees]]
- [[Sorting algorithms]] – Bubble, selection, insertion, merge, quick
- [[Searching algorithms]] – Linear, binary
- [[Database normalization]]
- [[OOP principles]] – Inheritance, polymorphism, encapsulation
