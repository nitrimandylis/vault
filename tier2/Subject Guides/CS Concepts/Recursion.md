# Recursion

## Definition
Function that calls itself to solve smaller instances of same problem.

## Components
- **Base case** — termination condition
- **Recursive case** — self-call with reduced input

## Examples
- Factorial: n! = n × (n-1)!
- Fibonacci: F(n) = F(n-1) + F(n-2)
- Tree traversal
- Binary search

## IB Exam Notes
- Often combined with linked lists and trees
- Know space complexity (O(n) call stack)
