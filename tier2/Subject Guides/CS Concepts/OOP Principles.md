# OOP Principles

## Encapsulation
- Bundle data + methods in class
- Access control: private, protected, public
- **Benefit**: Data hiding, modularity

## Inheritance
- Child class inherits parent attributes/methods
- **Benefit**: Code reuse, hierarchy

## Polymorphism
- Same interface, different implementations
- Method overriding (runtime)
- Method overloading (compile-time)

## Abstraction
- Hide complexity behind simple interface
- Abstract classes and interfaces

## IB Exam Focus
- Java implementation
- UML diagrams
- Choosing appropriate access modifiers
