# Searching Algorithms

## Linear Search
- Check each element sequentially
- O(n) — works on unsorted data

## Binary Search
- Divide sorted array in half each step
- O(log n) — requires sorted data
- **Must be sorted first** — consider sort cost

## IB Exam Tips
- Know when to use which
- Binary search: mid = (low + high) / 2
- Edge cases: empty array, target not found
