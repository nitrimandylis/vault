# CS IA – Project Hub

Computer Science Internal Assessment.

## Overview

The CS IA is a practical programming project (40 hours) that demonstrates understanding of computational thinking and programming concepts.

## Requirements

- Solve a real problem
- Must involve programming (Java or other)
- 40+ pages of documentation
- Must compile and run
