# Exam Strategy – CS HL

Exam preparation for CS HL.

## Papers Overview

### Paper 1 (2 hours)

- System fundamentals, networks, computational thinking
- Short answer and extended response

### Paper 2 (2 hours)

- Databases, object-oriented programming
- Program reading and writing

### Paper 3 (1 hour) – HL Only

- Option topic, code in context
- Short questions

## Study Approach

1. **Review core guide**: [[CS HL – Core Guide]]
2. **Practice past papers** – Under timed conditions
3. **Key topics**:
   - Big O complexity
   - Algorithm pseudo-code
   - Database queries (SQL)
   - OOP concepts

## Command Terms to Master

- Define, explain, describe
- Analyze, evaluate
- Construct, modify (code)

## Tips

- Know Java syntax well
- Practice writing pseudo-code
- Time management: ~1.5 min per mark
- Show working for partial credit
