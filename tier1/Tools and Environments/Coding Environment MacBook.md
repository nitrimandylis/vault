# Coding Environment – MacBook

## Hardware

- **Machine:** MacBook Pro M3 Pro
- **OS:** macOS (Apple Silicon)

## Terminal

- **Ghostty** — primary terminal emulator.
- CLI-first workflow. Always exploring new CLI tools.
- Git used from command line for all projects.

## Editors

- **VS Code** — daily driver. Extensions for Python, JS, markdown.
- **Zed** — speed and minimalism when VS Code is overkill.

## Theme

- Nord or Catppuccin across everything. Dark mode mandatory.

## Languages

- **Python** — most fluent. Used for scripting, AI/ML, automation.
- **JavaScript + HTML/CSS** — web front-end, school newspaper CMS.
- **Swift** — iOS apps (built before, occasional use).
- **AppleScript** — automation on macOS.

## Deployment

- **GitHub** — all projects. Public repos for portfolio.
- **Render** — web app deployment (e.g., IB Gazette site).
